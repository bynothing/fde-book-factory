# AI-FDE 图文线上发布版

本目录用于存放《AI-FDE：产业 AI 落地的新生产力体系》的图文移动阅读版产物。

## 当前产物

1. `AI-FDE-illustrated-online.html`：适合线上发布的独立 HTML 版本。
2. `AI-FDE-illustrated-mobile.pdf`：由同一 HTML 渲染出的移动端友好 PDF 版本，当前图版风格版共 283 页，约 1.49MB。
3. `preview-premium/`：本次出版图版风格调整后的封面、目录、图表、正文和移动端 HTML 截图预览。
4. `preview-redesign/`：上一轮重排版预览，保留用于对比。
5. `preview/`：早期图文版预览，保留用于对比。

## 生成命令

```powershell
node book-outline\scripts\build-illustrated-book.mjs
```

## 排版原则

1. HTML 面向移动端线上阅读，使用响应式布局。
2. PDF 面向移动端按页阅读，保留大字号、窄边距和全幅封面。
3. 图表不是装饰，而是章节进入前的结构化地图。
4. 每张图都包含图前楔子、图题、图体和图注。
5. 正文仍来自 `book-outline/manuscript/CH00-CH19`，不改写原始正文。
6. 当前图版风格版强调出版物气质：正文图示去掉圆角卡片、强阴影和泛 AI 高饱和配色，改用白底、细规则线、深灰蓝文字和少量墨绿色结构提示。
7. 图表在移动端采用纵向图版行结构，避免产品后台式模块堆叠；PDF 和 HTML 共用同一套生成脚本。
